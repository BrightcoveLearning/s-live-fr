Intégration d'une réunion Zoom dans Brightcove Live

L'intégration de Brightcove Live Zoom vous permet de diffuser facilement des réunions Zoom à l'aide de Brightcove Live.

Prérequis

Dans le Zoom Marketplace

L'intégration de Zoom à Brightcove Live est gérée par l'application Brightcove Live Connect sur Zoom Marketplace. Il s'agit d'une application de niveau compte destinée à être utilisée par des comptes Zoom de niveau propriétaire ou administrateur sous licence. Assurez-vous que votre compte Zoom dispose des privilèges appropriés pour prendre en charge l'intégration.

Dans Brightcove Video Cloud

Avant de démarrer une intégration, vous devez disposer d'un champ de métadonnées personnalisé pour savoir si un événement est diffusé en continu à partir de Zoom. Cela permettra de suivre l'état de l'intégration entre les sessions, c'est-à-dire si un événement est actuellement lié à une réunion Zoom. {% if site.product_short == Video Cloud-fr/ ou site.product_short == Beacon-fr/ ou site.product_short == CorpTV-fr/ %} Pour plus de détails sur la façon d'ajouter le champ metadata, veuillez consulter la documentation suivante. {% endif %} {% if site.product_short == Live-fr/ %} Pour plus de détails sur la façon d'ajouter le champ de métadonnées personnalisé, veuillez consulter la documentation suivante . {% endif %}

  1. Connectez-vous à Studio.
  2. Dans la navigation principale, cliquez sur Admin > Champs vidéo.
  3. Cliquez sur Ajouter un champ personnalisé.
  4. Saisissez un nom complet pour votre champ de métadonnées personnalisé.
  5. Pour le champ Nom interne, vous devez utiliser: bc_meeting_id
    N'ajoutez aucun ID ou donnée de zoom externe dans le champ personnalisé.
  6. Pour le Type, sélectionnez Texte.
  7. (Facultatif) Entrez une description pour le champ personnalisé.
  8. Cliquez sur Enregistrer le champ.

Meilleures pratiques

Pour de meilleurs résultats, veuillez suivre les instructions ci-dessous.

  1. Utilisez les boutons Démarrer et Arrêter la diffusion dans la salle de contrôle plutôt que sur le lecteur Zoom hôte.
  2. Évitez de diffuser plusieurs réunions Zoom en même temps sur un seul compte.

Création d'une réunion Zoom

  1. Tout d'abord, accédez à votre compte Zoom et créez une réunion Zoom. Pour plus d'informations sur la création d'une réunion Zoom, veuillez consulter la documentation Zoom.
  2. Dans Zoom, accédez à Réglages > Réunion (Avancé) > Réunions diffusées en direct.
  3. Assurez-vous que le service de diffusion en direct personnalisé est activé dans les paramètres de votre compte Zoom.

Intégration d'un compte Zoom à Brightcove Live

Pour intégrer un compte Zoom à Brightcove Live, procédez comme suit:

  1. Connectez-vous à Studio.
  2. Dans la navigation principale, cliquez sur Admin > Intégrations.
  3. Sélectionnez l'option Intégrations dans le menu déroulant.
  4. Cliquez sur le bouton Ajouter une intégration pour associer un ou plusieurs comptes Zoom à votre compte Brightcove.
  5. Une fenêtre contextuelle vous demandant d'accéder à votre compte Zoom s'affiche. Cliquez sur le bouton Autoriser.
  6. Fournissez vos informations d'identification Zoom pour intégrer le compte.

Ajout de l'intégration Zoom à un événement Brightcove Live

Pour ajouter l'intégration Zoom à l'événement Brightcove Live, procédez comme suit:

  1. Dans Studio, ouvrez le module Live.
  2. Cliquez sur le bouton Créer un événement sur la page Gérer les événements en direct. Pour plus d'informations sur la création d'un événement en direct, veuillez consulter la documentation Création et gestion d'événements en direct à l'aide du module Live.
  3. Remplissez les détails de l'événement en direct comme d'habitude, jusqu'à ce que vous atteigniez la section Intégrations.
  4. Cliquez sur le bouton Ajouter une intégration.
  5. Vous pourrez voir la liste de vos réunions Zoom actives et planifiées.
  6. Cliquez sur le menu déroulant.
  7. Sélectionnez votre réunion. Dans cet exemple, je choisis la réunion Test Docs.
    Remarque: Une seule intégration peut être ajoutée à un événement à la fois. Pour effacer l'intégration, cliquez sur le bouton Supprimer l'intégration. Si vous souhaitez avoir plus d'un compte, vous pourrez le faire dans le panneau Admin > Intégrations.
  8. Une fois que vous êtes prêt à créer l'événement en direct, cliquez sur le bouton Créer un événement.
  9. Dans la Control Room, une option à onglets vous permet de diffuser votre réunion en streaming depuis Zoom. Si vous préférez ne pas utiliser la source d'intégration, revenez à l'onglet Ingest URL dans la partie supérieure gauche.
  10. Si votre réunion Zoom a été récemment créée, essayez de la synchroniser à nouveau avec Zoom à l'aide du bouton Synchroniser.
    Si votre réunion ne s'affiche toujours pas après la synchronisation, essayez de vérifier que votre réunion est planifiée dans Zoom et que vous avez moins de 30 réunions planifiées.
  11. Revenez à votre compte de réunion Zoom.
  12. Cliquez sur le bouton bleu Démarrer pour démarrer votre réunion.
    Remarque: En raison d'une limitation de Zoom, avant de commencer à diffuser la réunion Zoom sur Brightcove Live, la réunion Zoom doit être lancée.
  13. De retour dans la salle de contrôle en direct, cliquez sur le bouton Démarrer la diffusion.
  14. Après la dernière étape, vous pourrez voir votre réunion Zoom dans la Control Room.
    • À tout moment pendant la réunion Zoom, vous pouvez arrêter et relancer la diffusion. Toutefois, si vous ne diffusez pas activement l'événement peut prendre fin si l'écart dépasse le temps de reconnexion de l'événement.
    • La diffusion de Zoom sur Brightcove sera retardée d'environ 20 secondes.
    • Si la réunion Zoom ne démarre pas après 20 secondes, vous pouvez cliquer sur Arrêter la diffusion, puis sur Redémarrer la diffusion.
    • Le flux Zoom étant un service tiers, certaines fonctionnalités Live avancées peuvent ne pas être prises en charge, y compris SSAI.

Désinstallation de l'intégration Zoom

Suppression de l'intégration

Pour supprimer les intégrations Zoom de votre compte:

  1. Connectez-vous à Studio.
  2. Dans la navigation principale, cliquez sur Admin > Intégrations.
  3. Cliquez sur l'icône Corbeille.
  4. Lorsque vous êtes invité à confirmer, cliquez sur le bouton Confirmer.

Pour supprimer Brightcove Live Connect de votre compte Zoom

Vous pouvez supprimer complètement l'application Brightcove Live Connect de votre compte Zoom en vous connectant à Zoom:

  1. Connectez-vous à votre compte Zoom et accédez à Zoom App Marketplace.
  2. Cliquez sur Gérer > Applications installées ou recherchez l'application Brightcove.
  3. Cliquez sur l'application Brightcove.
  4. Cliquez sur Désinstaller.
