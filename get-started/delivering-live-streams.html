La fonction de diffusion vidéo en direct de Brightcove vous permet de diffuser des diffusions en direct de haute qualité sur n'importe quel appareil ou navigateur. Vous pouvez diffuser des flux en direct sans développement personnalisé ni intégration complexe. Toutefois, des API sont disponibles si vous avez des besoins personnalisés ou si vous avez besoin d'intégrations personnalisées.

Remarque: La prise en charge du streaming en direct est disponible uniquement pour les éditeurs Enterprise.

Brightcove prend également en charge la diffusion en direct avec DVR Lorsqu'un lecteur Brightcove est configuré avec le plug-in DVRUX, les flux en direct se comportent comme s'ils étaient lus via l'enregistreur vidéo numérique du spectateur.

Brightcove propose plusieurs options pour diffuser des flux en direct. Cette rubrique contient les sections suivantes:

  • Encodage matériel ou basé sur le cloud
  • Diffusion en direct à l'aide du module Live
  • Diffusion en direct à l'aide d'un encodeur matériel

Encodage matériel ou basé sur le cloud

Brightcove prend en charge l'encodage matériel et basé sur le cloud. Divers facteurs peuvent déterminer si le codage matériel ou basé sur le cloud convient le mieux à un événement.

Quand utiliser le codage matériel

  • Avances linéaires 24 h/24, 7 j/7 lorsque la capacité est connue
  • Une architecture basée sur un seul site
  • Support des formats de diffusion

Quand utiliser l'encodage dans le cloud

  • Pour les événements
  • Lorsque les besoins en capacité sont inconnus
  • Pour les organisations décentralisées
  • Lorsque le moment de l'événement est inconnu et que la vitesse est importante
  • S'il existe des goulots d'étranglement de la bande passante

Diffusion en direct à l'aide du module Live

Le module Live fournit une interface facile à utiliser pour vous aider à diffuser un événement en direct sur les ordinateurs de bureau et les appareils mobiles. Aucune programmation n'est requise. Après avoir saisi les informations de votre événement dans le module Live, vous recevrez un ensemble de paramètres d'encodeur que vous connecterez à votre encodeur sur site. Tout le transcodage est effectué dans le cloud, vous n'avez donc pas besoin d'un transcodeur matériel. Pour plus d'informations, consultez l'un de ces documents.

API Live

L'API Live est une API basée sur REST qui vous permet de créer et de gérer des événements de diffusion en direct. Les fonctionnalités optionnelles incluent:

  • Insertion de publicités côté serveur (SSAI)
  • Chiffrement AES-128
  • Créez des ressources vidéo à la demande à partir de clips extraits de la diffusion en direct
  • Capacité DVR
  • Plusieurs CDN

Pour plus d'informations, consultez la documentation de l'API Live.

Diffusion en direct à l'aide d'un encodeur matériel

Cette option utilise un encodeur matériel pour encoder et diffuser des flux en direct sur votre CDN. Du côté de Video Cloud, vous créez des ressources distantes pour héberger les flux, puis vous pouvez publier un lecteur pour votre flux en direct.
