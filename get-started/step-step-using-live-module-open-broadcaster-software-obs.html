Ce Quick Start vous aidera à diffuser un événement en direct sur différents appareils à l'aide d'Open Broadcaster Software (OBS) Studio. OBS est un logiciel open source pour l'enregistrement vidéo et la diffusion en direct et peut être téléchargé ici. D'autres encodeurs peuvent également être utilisés avec le module Live.

Public

Tout éditeur ayant besoin de diffuser un événement de diffusion en direct à l'aide du module Live.

Prérequis

  • Un compte Enterprise avec accès au module Live
  • Une caméra pour diffuser l'événement
  • Vous devez disposer de votre propre logiciel d'encodage ; ce Quick Start utilise OBS Studio qui peut être téléchargé gratuitement

Limitations

  • Votre encodeur doit émettre un flux RTMP ou SRT
  • Les diffusions en direct connaîtront un délai de 20 à 30 secondes
  • La diffusion en continu doit commencer dans les 30 minutes suivant le début de l'événement (paramètre de compte par défaut)

Étapes à suivre pour diffuser un événement en direct

Pour diffuser votre événement en direct à l'aide du module Live, vous devez effectuer les tâches suivantes:

  • Créer un nouvel événement à l'aide du module Live
  • Configuration de votre encodeur
  • Publiez la diffusion en direct
  • Créez un extrait de l'événement en direct

Créer un nouvel événement en direct

  1. Connectez-vous à Studio.
  2. Ouvrez le module Live.
  3. Cliquez sur le bouton Créer un événement pour créer un nouvel événement.
  4. Saisissez un nom d'événement.
  5. Saisissez quelques balises pour l'événement.
  6. Si votre compte a été configuré avec des champs personnalisés obligatoires, assurez-vous que des valeurs sont attribuées aux champs personnalisés.
  7. Sélectionnez un profil d'ingestion en direct dans la liste déroulante. Le profil d'ingestion en direct contrôle le nombre et la qualité des rendus en direct créés.
  8. Sélectionnez Convertir l'événement en vidéo lorsque vous avez terminé. Cela enregistrera l'événement sous forme de vidéo dans le module Médias.
  9. Dans la section Options de diffusion, sélectionnez la région la plus proche de votre emplacement.
  10. Dans la section Options avancées, définissez le temps de reconnexion sur 10 minutes.
    Remarque: OBS prend en charge le protocole SRT et peut être sélectionné à l'aide de la section Format d'entrée du menu Options avancées.
  11. Cliquez sur Créer un événement. Le module Live renvoie un point de terminaison de flux (URL RTMP) et un nom de flux. Ces valeurs seront utilisées dans les paramètres de l'encodeur.
    Remarque: Si vous avez sélectionné le protocole SRT pour l'événement Live, celui-ci sera disponible dans OBS en choisissant Personnalisé-fr/ sous Stream settings Service, puis en saisissant votre chemin srt://dans la zone Serveur.

Configuration de votre encodeur

Dans cette section, nous allons configurer et utiliser l'encodeur OBS Studio pour prendre en charge un événement de diffusion en direct. Le logiciel OBS Studio est une application de bureau qui capture les entrées de votre caméra et produit un flux qui peut être diffusé par un CDN. Notez qu'il existe des solutions d'encodage matérielles et logicielles qui peuvent être mieux adaptées à la diffusion de votre événement de diffusion en direct. Vous pouvez télécharger le logiciel OBS Studio ici.

Remarques:
  • Le logiciel OBS doit être installé et une caméra connectée à votre ordinateur avant de poursuivre.
  • Cet exemple utilise un appareil photo Canon connecté à un ordinateur portable à l'aide du logiciel Canon EOS Webcam Utility.
  • OBS offre la possibilité d'enregistrer une copie locale du flux. En choisissant cette option, vous aurez une copie locale du flux à partir de laquelle vous pourrez créer des clips au cas où des problèmes de réseau se produiraient pendant la diffusion de l'événement. Pour plus d'informations sur la façon d'enregistrer un fichier local, consultez la documentation OBS.
  1. Ouvrez OBS Studio.
  2. Dans la section Sources, cliquez sur + pour ajouter une nouvelle source. Sélectionnez une source dans la liste. Ce Quick Start utilise une caméra pour la capture vidéo, de sorte que le périphérique de capture vidéo est sélectionné.
  3. Entrez un nom pour la nouvelle source et cliquez sur OK.
  4. Sélectionnez votre appareil photo dans la liste des appareils. Un aperçu doit apparaître dans la boîte de dialogue des propriétés. Cliquez sur OK.
  5. Dans la section Contrôles, cliquez sur Paramètres.
  6. Cliquez sur Stream dans la partie gauche de la boîte de dialogue Paramètres.
  7. Pour Stream Service, sélectionnez Custom... .
  8. Pour le serveur, utilisez l'URL du flux fournie par le module Live.
  9. Pour la Stream Key, utilisez alive.
  10. Cliquez sur OK pour enregistrer les paramètres.
  11. Dans la section Contrôles, cliquez sur Démarrer la diffusion en continu. La diffusion en direct devrait commencer.
Remarque: Le traitement de vos flux prendra environ une minute.

Publier l'événement en direct

Pour générer le code d'intégration de l'événement en direct, procédez comme suit:

  1. Revenez au module Live.
  2. Si la page Control Room de l'événement n'est pas affichée, cliquez sur l'onglet Live pour afficher la liste des événements en direct. Cliquez sur le nom de l'événement pour ouvrir la page Control Room.
  3. Vérifiez que vous pouvez voir la diffusion en direct. Notez qu'après le démarrage de l'encodeur, la diffusion en direct peut prendre une minute.
  4. Pour publier l'événement en direct, cliquez sur Publier et intégrer dans le volet de navigation de gauche.
  5. Sélectionnez un lecteur à l'aide de la liste déroulante Available Player.
  6. Cliquez sur l'URL du lecteur pour visionner la diffusion en direct.
  7. Lorsque vous avez terminé la diffusion en direct, cliquez sur le bouton Arrêter ce flux sur la page Control Room. Revenez également dans OBS Studio et cliquez sur le bouton Arrêter la diffusion en continu pour arrêter la diffusion en continu de l'encodeur.
    Remarque: Il est important de vous assurer de fermer/arrêter votre encodeur lorsque la diffusion en direct est terminée. Le fait de laisser une connexion ouverte entre n'importe quel encodeur et Brightcove, même lorsqu'aucune donnée n'est envoyée, peut entraîner des frais de connexion imprévus.

Création d'un clip

Le module Live permet de créer des clips vidéo à partir d'événements terminés. Des clips peuvent également être créés pendant qu'un événement en direct est en cours. Plusieurs raisons peuvent vous inciter à créer un clip:

  • Pour créer un court clip d'accroche pouvant être publié sur les réseaux sociaux
  • Pour limiter la longueur d'une vidéo que vous souhaitez publier (la plupart des plateformes limitent la longueur des vidéos)
  • Vous avez constaté que les vidéos plus courtes sont plus performantes sur les réseaux sociaux

Les clips ne peuvent être créés que dans les 7 jours suivant la fin de l'événement. Ensuite, le bouton Créer un clip sera désactivé. Si l'événement dure plus de 24 heures, seules les dernières 24 heures peuvent être découpées.

Pour créer un clip, procédez comme suit:

  1. Revenez au module Live.
  2. Cliquez sur l'onglet Terminé.
  3. Localisez l'événement et cliquez sur son nom pour ouvrir la Control Room.
  4. Dans le volet de navigation de gauche, cliquez sur Créer un clip. L'éditeur d'écrêtage s'ouvre.
  5. Cliquez et faites glisser les poignées de découpe sur la chronologie de la mise au point pour définir les points de départ et de fin du clip.
  6. Cliquez sur Aperçu du clip pour prévisualiser le clip. Les points de départ et d'arrivée peuvent être ajustés selon les besoins.
  7. Cliquez sur Enregistrer.
  8. Entrez un nom de clip et d'autres valeurs de métadonnées. Tous les champs obligatoires pour le compte apparaîtront également et des valeurs doivent être attribuées.
  9. Cliquez sur Créer un clip. Le processus d'ingestion commencera et le clip sera enregistré sur votre compte.
  10. Les clips apparaîtront sous l'éditeur de clips. Cliquez sur Publier sur les réseaux sociaux pour publier le clip sur l'une de vos destinations sociales à l'aide de Brightcove Social. Pour plus d'informations sur la publication de vidéos à l'aide du module Social, consultez {% if site.product_short == Video Cloud-fr/ ou site.product_short == Beacon-fr/ ou site.product_short == CorpTV-fr/ %} Publier des vidéos sur les plateformes de réseaux sociaux. {% endif %} {% if site.product_short == Live-fr/ %} Publier des vidéos sur les plateformes de réseaux sociaux. {% endif %}

Des clips apparaîtront également dans la Control Room pour l'événement.

Remarque: Vous pouvez cliquer plusieurs fois sur le bouton Créer un clip pour créer plusieurs clips de votre diffusion. Chaque clip est enregistré en tant que ressource VOD distincte dans le module Médias.

Résolution des problèmes de diffusion en direct

Si vous avez besoin d'une aide supplémentaire pour que votre événement en direct fonctionne, vous pouvez nous contacter. Pour vous assurer d'obtenir une réponse la plus rapide possible, vous trouverez ci-dessous une liste de l'assistance dont vous aurez besoin pour résoudre le problème.

  • Les symptômes spécifiques du flux. Par exemple, est-ce qu'il ne joue pas du tout ou est-ce qu'il bégaie ou gèle ?
  • Si ce flux fonctionnait correctement par le passé
  • L'URL du point d'entrée que vous utilisez dans votre encodeur
  • Le logiciel et le matériel d'encodage que vous utilisez
  • L'URL du lecteur sur lequel vous avez publié l'événement en direct
  • L'identifiant vidéo de votre actif en direct
  • Les résultats d'un trace-route de votre encodeur vers l'hôte du point de publication
